## Visualize Data with a Heat Map
### Part of the freeCodeCamp curriculum

### User Stories
- I can view a heat map with data represented both on the Y and X axis.
- Each cell is coloured based on its relationship to other data.
- I can mouse over a cell in the heat map to get more exact information.

### Running on [Heroku](http://andydlindsay-heatmap.herokuapp.com)
